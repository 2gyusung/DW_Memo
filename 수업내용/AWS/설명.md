### S3(Simple Storage Service)
```

1. 이미지 영상 전용 사용
2. 백업 서버 사용
3. 웹 호스팅[웹사이트 임대] 서버 사용(✔)
    > 임대 해서 HTML을 S3에 업로드

특징 : 저장공간 무한이다.
가격 : 시간당 저장 용량 만큼 가격 책정

-   5GB 이하는 1년동안 공짜

```

동적웹 : 프레임워크 + 템플릿 엔진 
        Spring +  JSP   
정적웹 :  HTML , JS , JQuery

차이점 : DB를 연동 할 수 있는지 없는지 차이

### RDS(Relational Database Service)
```
AWS에서 제공하는 데이터베이스 전용 서버

퍼블릭 엑세스를 설정 해야 접속
```
